# Facial Emotion Recognition using CNN

## 📌 Project Overview
This project focuses on building a facial emotion recognition system using image processing and deep learning techniques. The system analyzes facial images and classifies human emotions into different categories such as happy, sad, angry, fearful, neutral, surprised, and disgusted.

---

## 🎯 Objective
The main goal of this project is to develop a classification model that can accurately detect emotions from facial images using Convolutional Neural Networks (CNN).

---

## 📊 Dataset
- Dataset: FER2013
- Source: Kaggle
- Number of images: 35,887
- Image size: 48 × 48 (grayscale)
- Classes: 7 emotions  
  (angry, disgust, fear, happy, neutral, sad, surprise)

---

## ⚙️ Preprocessing Techniques
The following preprocessing techniques are applied to improve model performance:

- **Normalization**: Scaling pixel values from [0,255] to [0,1]
- **Histogram Equalization**: Enhancing image contrast
- **Data Augmentation**: Increasing dataset diversity (rotation, flip, zoom)
- **Noise Reduction**: Removing unwanted noise using filters
